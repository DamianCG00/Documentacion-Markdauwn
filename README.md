# Portafolio de Actividades - Calculadoras 

## Información del Proyecto

Este portafolio integra diversas actividades interactivas de HTML, CSS y JavaScript, diseñadas para demostrar conceptos matemáticos de forma dinámica y visual. Cada actividad está disponible directamente en la página principal mediante "cartillas", permitiendo al usuario interactuar con ellas sin necesidad de abrir múltiples archivos.

## Actividades Integradas

El portafolio incluye las siguientes actividades:

1. *Actividad #2:* Propiedades numeros complejos   
2. *Actividad #4:* Forma polar a forma exponencial  
3. *Actividad #6:* Expresiones algebraicas  
4. *Actividad #8:* Calculadora operaciones
5. *Actividad #10:* Productos notables  
6. *Actividad #12:* División Sintética de Polinomios
7. *Actividad #14:* Simbólica de Ecuaciones

Cada actividad está alojada en su propia carpeta actividad02, actividad04, etc.

## Funcionalidad

- Cada actividad se puede visualizar directamente en la página principal mediante fichas.
- Los contenedores están diseñados para ser dinámicos y centrados.

## Tecnologías Utilizadas

- HTML5
- CSS
- JavaScript

## Cómo usar el portafolio

1. Clonar el repositorio o descargarlo como ZIP.
2. Abrir el archivo index.html en un navegador moderno.
3. Hacer clic en cada actividad para interactuar con las calculadoras y animaciones.
